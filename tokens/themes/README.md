# Themes

This directory contains theme definitions for the Gonzalo Herrera Brand System.

Themes are built using semantic tokens and define how the visual system is applied in different environments.

## Available Themes

### Light

Primary theme used across:

* Website
* LinkedIn Assets
* Portfolio
* Documentation

### Dark

Secondary theme optimized for:

* Presentations
* Developer-oriented content
* Future dark mode support

## Principles

Themes should reference semantic tokens whenever possible.

Avoid hardcoding primitive colors directly into components.
